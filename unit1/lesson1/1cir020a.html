Course Purpose and Navigation Features - Navigation Tutorial Part I

Narrator: As you complete this training, you will be working through scenarios that are set in several different locations all over the world. Let’s take a look at unit 3, OEHSA, which is set at Nigel AFB in North Pole, Alaska.  You may notice that the base names don’t correspond with actual bases.  These names have been invented for the purpose of this course.

The lesson menu shows you the lessons that are in the unit.  The number of lessons per unit varies.  For example, some units have only one lesson, but other units may have up to six lessons.  By rolling over each lesson title, you can read a brief overview of what you will learn in that lesson.  You can enter the lesson by selecting the lesson title.

The BESA I course has been designed with several features that will help you get the most out of your training experience.  First, let’s look at the static elements of the screen – or the parts that remain the same throughout each lesson. 

At the top left of each screen you’ll find the course name, unit title, and lesson title.  On the top right, you’ll find the page number, as well as a map that serves as a reminder about the setting of the unit. Remember, this unit is set in Alaska.  Notice the snow across the top of the screen?  This is a graphic element which will change depending on your location.

Now let’s look at the bottom of the screen.  You’re already familiar with the Forward button, which takes you to the next page.  Selecting the Repeat button refreshes the screen.  This is useful if you want to view an animation again or if you need to reset information.  And of course, selecting Previous will go back to the previous page.

Across the bottom of every screen is the prompt area, which you’ve already learned about.  Although the prompt area is always there, the text in the prompt will be different for each screen.
