Narrator: Let’s go into another unit to look at some more features of the course.

This unit, which is about Potable Water, is set at a bare base in Chad, Africa.

Notice that the map at the top of the screen shows this location, and the border across the top has changed to sand.

The Lesson Map is another tool you can use to navigate the course, in addition to the Forward and Previous buttons.

With the Lesson Map, you can easily jump to another page in the lesson.  You may notice that not all of the pages in the lesson are linked in the Lesson Map.  You’ll learn more about this later.  Right now, let’s take a look at the lesson summary page.

You’ll notice that the last page of each lesson is similar to the first one.  The text provides a summary of what you learned in the lesson, and the graphic allows you to view your progress in the unit.

Selecting Forward from this page will return you to the lesson menu so you can select the next lesson.

You’ll learn more about the features in this course in the rest of the introductory lesson.
