Probe Burnout Calculation

The probe burnout calculation is used when monitoring pulsed wave systems to calculate the maximum power density to which the probe can be exposed prior to burnout (Smax).  The formula used to calculate probe burnout is:

  Smax = duty factor  x  burnout rating
probe correction factor

The burnout rating can be found in the manufacturer’s literature.  The probe correction factor is located on the probe’s handle and corresponds to frequency.

If the calculated value is above the full scale value of the meter, the likelihood of burnout is low.  If the calculated value is below the full scale value of the meter, the likelihood of burnout is high.
