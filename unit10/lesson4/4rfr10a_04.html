Exposed Within Far-Field

If the individual was exposed outside the near-field range of the emitter, you should calculate the power density using the far-field power density calculation (Sff).  The far-field power density can be calculated using the following formula:

  Sff = (Pavg) (Gabs)

Where:
Gabs = absolute gain
Pavg = average power

Where:
Qs= scan angle in radians
B = half-power beam width
